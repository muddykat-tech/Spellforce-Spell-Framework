Initializes the Spellforce Spell Framework (SFSF). 

This function initializes the Spellforce Spell Framework (SFSF), setting up hooks, APIs, vanilla spells, and loading/registering mods.

The initialization process includes the following phases\+:


\begin{DoxyItemize}
\item {\bfseries{Framework Configuration Phase}}\+:
\begin{DoxyItemize}
\item {\bfseries{Data Hook Initialization}}\+: Sets up low-\/level hooks to the game\textquotesingle{}s memory and functions.
\item {\bfseries{API Function Linking}}\+: Links framework API structures.
\item {\bfseries{Vanilla Spell Initialization}}\+: Registers vanilla spell types and effect handlers.
\end{DoxyItemize}
\item {\bfseries{Mod Loading Phase}}\+:
\begin{DoxyItemize}
\item Loads all mods compatible with the framework.
\end{DoxyItemize}
\item {\bfseries{Mod Registration Phase}}\+:
\begin{DoxyItemize}
\item Registers spells and functions from loaded mods. \begin{DoxyWarning}{Warning}
Incorrect setup of hooks or APIs can lead to crashes. Mods must adhere to the framework\textquotesingle{}s API for compatibility. 
\end{DoxyWarning}
